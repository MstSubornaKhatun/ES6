
Module 1-1: ES6 Intro – var, let, const

Cue (কী-ওয়ার্ড / প্রশ্ন)
- JavaScript এর ইতিহাস কী?
- ES6 এ var, let, const এর মধ্যে পার্থক্য কী?
- কোনটা কখন ব্যবহার করা উচিত?
- const দিয়ে array/object কিভাবে কাজ করে?

Notes (Detailed Information)

1. History of JavaScript
- JavaScript তৈরি হয় 1995 সালে Brendan Eich দ্বারা।
- প্রথমে Netscape browser এর জন্য scripting language হিসেবে।
- পরে ECMAScript standard হয়।
- ES6 (2015) → বড় update, নতুন feature যেমন let, const, arrow function, class, template literals ইত্যাদি।

2. Difference between var, let, const
- var → function scoped, re-declare করা যায়, hoisting হয়।
- let → block scoped, re-assign করা যায়, re-declare করা যায় না।
- const → block scoped, re-assign করা যায় না, কিন্তু array/object এর ভিতরের মান পরিবর্তন করা যায়।

3. Code Explanation (লাইন-বাই-লাইন)

console.log(name); // error
Explanation: const দিয়ে declare করার আগে access করলে error হবে (Temporal Dead Zone)।

console.log('device value ', device ); // device value undefined
var device = 'laptop';
Explanation: var hoisting হয়। declare করার আগে access করলে undefined পাওয়া যায়।

const name = 'Bangladesh';
// name = 'BD' // error
Explanation: const দিয়ে declare করলে re-assign করা যায় না। তাই error হবে।

const countryName = 'BD' + name; 
console.log(countryName); // BDBangladesh
Explanation: const দিয়ে নতুন variable তৈরি করা যায় এবং value assign করা যায়। এখানে string concatenation হয়েছে।

let price = 130;
price = 150;
console.log(price); // 150
Explanation: let দিয়ে declare করলে re-assign করা যায়। তাই price এর মান 150 হয়েছে।

const charge = 50;
// charge = 45;
Explanation: const দিয়ে declare করলে re-assign করা যায় না। তাই error হবে।

const dress = ['shirt', 'pant', 'jacket']
// dress = []; // error
dress.push('maflar');
console.log(dress); // [ 'shirt', 'pant', 'jacket', 'maflar' ]
Explanation: const দিয়ে array declare করলে পুরো array re-assign করা যায় না। কিন্তু push/pop দিয়ে ভিতরের মান পরিবর্তন করা যায়।

const student = {name: 'jabbar', marks: 50}
student.marks = 60;
console.log(student); // { name: 'jabbar', marks: 60 }
Explanation: const দিয়ে object declare করলে object re-assign করা যায় না। কিন্তু property পরিবর্তন করা যায়।

Output (Example)
- var → hoisting হয়, function scoped।
- let → block scoped, re-assign করা যায়।
- const → block scoped, re-assign করা যায় না, কিন্তু array/object এর ভিতরের মান পরিবর্তন করা যায়।

Summary
- JavaScript history → ES6 এ বড় update।
- var → পুরনো, hoisting হয়।
- let → modern, block scoped।
- const → constant value, কিন্তু array/object mutable।
- Module 1-1 → ES6 intro এবং var, let, const এর পার্থক্য।


Module 1-2: Function Default Parameter (=)

Cue (কী-ওয়ার্ড / প্রশ্ন)
- Argument কী?
- Parameter কী?
- Default parameter কীভাবে কাজ করে?
- কেন default parameter দরকার?
- NaN (Not a Number) কখন আসে?

Notes (Detailed Information)

1. Argument vs Parameter
- Parameter → function বানানোর সময় ভিতরে যে নামগুলো লিখি।
  উদাহরণ: function sum(num1, num2) { ... }
  এখানে num1, num2 হলো parameter।

- Argument → function কল করার সময় যে মানগুলো দিই।
  উদাহরণ: sum(10, 20)
  এখানে 10, 20 হলো argument।

2. Default Parameter Concept
- যদি function কল করার সময় কোনো argument না দিই,
  তাহলে parameter এর জন্য একটা default মান ব্যবহার হয়।
- এতে করে undefined বা NaN আসে না।
- সাধারণ default মান:
  add, subtract → 0  
  multiply → 1  
  string → ''  
  array → []  
  object → {}  

3. Code Explanation (লাইন-বাই-লাইন)

function sum(num1 = 0, num2 = 0){
    const total = num1 + num2;
    console.log(num1, num2, total);
}

sum(10, 20) → Output: 10 20 30  
Explanation: num1=10, num2=20 → total=30  

sum(10) → Output: 10 0 10  
Explanation: num1=10, num2 default=0 → total=10  

sum(10, 20, 40, 50) → Output: 30  
Explanation: অতিরিক্ত argument ignore হয়। শুধু প্রথম দুইটা নেয়।  

Without default parameter:  
sum(10) → num2 undefined → total=NaN  

---

function multiply(num1=1, num2=1) {
    const result = num1 * num2;
    console.log(result);
}

multiply(2, 3) → Output: 6  
Explanation: num1=2, num2=3 → result=6  

multiply(2) → Output: 2  
Explanation: num1=2, num2 default=1 → result=2  
(যদি default না থাকত, num2 undefined → NaN)  

---

function fullName(first='', last='') {
    const name = first + '' + last;
    console.log(name);
}

fullName('kamal') → Output: kamal  
Explanation: first='kamal', last default='' → name='kamal'  

fullName('kamal', 'uddin') → Output: kamaluddin  
Explanation: দুইটা argument দিলে full name তৈরি হয়।  

Without default parameter:  
fullName('kamal') → Output: kamalundefined  

Output (Example)
- sum(10,20) → 30
- sum(10) → 10 (default parameter কাজ করছে)
- multiply(2,3) → 6
- multiply(2) → 2 (default parameter কাজ করছে)
- fullName('kamal') → kamal (default parameter কাজ করছে)

Summary
- Parameter: function এর ভিতরে নাম।
- Argument: function কল করার সময় দেওয়া মান।
- Default parameter: argument না দিলে fallback মান ব্যবহার হয়।
- NaN আসে যখন undefined এর সাথে calculation হয়।
- Module 1-2 → Function default parameter এর ব্যবহার এবং সুবিধা।


Module 1-3: Template String, Multiple Line String, Dynamic String (` `)

Cue (কী-ওয়ার্ড / প্রশ্ন)
- Template string কী?
- Multiple line string কিভাবে লেখা যায়?
- Dynamic string কীভাবে কাজ করে?
- \n, '', "", `` এগুলোর কাজ কী?

Notes (বিস্তারিত তথ্য)

1. Template String Concept
- Template string হলো backtick (` `) দিয়ে লেখা string।
- এর মাধ্যমে:
  → একাধিক লাইন একসাথে লেখা যায়।
  → Dynamic মান (variable) string এর মধ্যে বসানো যায়।
- আগে string বানাতে + দিয়ে join করতে হতো।
- এখন সহজে ${variable} ব্যবহার করে dynamic string বানানো যায়।

2. Common Escape Characters
- \n → নতুন লাইন।
- '' বা "" → সাধারণ string।
- `` → template string (ES6 থেকে এসেছে)।
- ${} → dynamic মান বসানোর জন্য।

Code Example + Explanation

// Wrong way (error হবে)
const poem = 'amm pata jora jora
marbu chabuk chorbe ghora'

Explanation: এখানে single quote (' ') দিয়ে multi-line লিখলে error হবে।

---

const poem = 'amm pata jora jora \n' +
'marbu chabuk chorbe ghora\n' +
'ore khoka ghure dara'

Explanation: এখানে \n দিয়ে নতুন লাইন তৈরি করা হয়েছে। কিন্তু অনেক ঝামেলা, প্রতিটি লাইনে + দিতে হচ্ছে।

---

const poem = `Amm pata jora jora
marbo chabuk chorbe ghora
ore babu ghure dara
asche amar pagla ghora`;
console.log(poem)

Explanation: এখানে backtick (` `) ব্যবহার করা হয়েছে।
- একাধিক লাইন একসাথে লেখা গেছে।
- কোনো \n দরকার হয়নি।
- Output সুন্দরভাবে multiple line এ আসবে।

---

function sum(num1, num2){
    const result = num1 + num2;
    // const output = 'sum of ' + num1 + ' and ' + num2 + ' is equal to ' + result;
    const output = `sum of ${num1} and ${num2} is equal to ${result}`
    console.log(output)
}
sum(5,7)

Explanation:
- আগের style: 'sum of ' + num1 + ' and ' + num2 ...
- নতুন style: `sum of ${num1} and ${num2} is equal to ${result}`
- এখানে ${variable} দিয়ে dynamic মান বসানো হয়েছে।
- Output: sum of 5 and 7 is equal to 12

---

const price = 50;
const discountedPrice = `Discounted Price of ${price} after 20% discount ${price * 0.8}`
console.log(discountedPrice)

Explanation:
- price = 50
- ${price * 0.8} → dynamic calculation
- Output: Discounted Price of 50 after 20% discount 40

Output (Example)
- Template string দিয়ে multiple line লেখা যায়।
- Dynamic মান বসানো যায় ${variable} দিয়ে।
- Calculation বা expressionও বসানো যায়।
- Output clean এবং readable হয়।

Summary
- Template string → backtick (` `) দিয়ে লেখা হয়।
- Multiple line string → সহজে লেখা যায়, \n দরকার নেই।
- Dynamic string → ${variable} দিয়ে মান বসানো যায়।
- Expressionও বসানো যায় → ${price * 0.8}
- Module 1-3 → Template string এর ব্যবহার, multiple line, dynamic string।


Module 1-4: Arrow Function, Multiple Parameters, Function Body (=>)

Cue (কী-ওয়ার্ড / প্রশ্ন)
- Arrow function কী?
- Function declaration, function expression আর arrow function এর মধ্যে পার্থক্য কী?
- Multiple parameters কিভাবে handle করা হয়?
- Multi-line arrow function কিভাবে লেখা হয়?

Notes (বিস্তারিত তথ্য)

1. Arrow Function Concept
- Arrow function হলো ES6 (ECMAScript 2015) এ আসা নতুন feature।
- Syntax: const funcName = (param1, param2) => expression;
- সুবিধা:
  → ছোট এবং clean syntax।
  → এক লাইনে function লেখা যায়।
  → Multiple parameters handle করা যায়।
  → Multi-line body থাকলে {} ব্যবহার করতে হয় এবং return লিখতে হয়।

2. Function Types
- Function declaration → function add(num1, num2) { return num1 + num2; }
- Function expression → const addition = function(num1, num2) { return num1 + num2; }
- Arrow function → const add2 = (num1, num2) => num1 + num2;

Code Example + Explanation

// Function declaration
function add(num1, num2) {
    return num1 + num2;
}
Explanation: সাধারণভাবে function বানানো। num1+num2 return করছে।

// Function expression
const addition = function (num1, num2) {
    return num1 + num2;
}
Explanation: function কে একটি variable এ assign করা হয়েছে। একই কাজ করছে।

// Arrow function
const add2 = (num1, num2) => num1 + num2;
Explanation: ছোট syntax। num1+num2 return করছে।

const multiply = (a, b) => a * b;
Explanation: দুইটা parameter নিয়ে গুণ করছে।

const sumAll = (a, b, c, d, e, f) => a + b + c + d + e + f;
Explanation: একাধিক parameter নিয়ে সব যোগ করছে।

const isFirstBig = (x, y) => x > y;
Explanation: দুইটা সংখ্যা compare করছে। true/false return করছে।

---

// Multi-line arrow function
const doMath = (x, y) => {
    const makeDouble = x * 2;
    const againDouble = y * 2;
    const result = makeDouble + againDouble;
    return result;
}
Explanation:
- এখানে {} ব্যবহার করা হয়েছে।
- একাধিক statement আছে।
- শেষে return দিয়ে result পাঠানো হয়েছে।

---

// Function call examples
const result = add2(17, 13);
console.log(result) // 30
Explanation: add2(17,13) → 30

const mult = multiply(5, 7);
console.log(mult); // 35
Explanation: multiply(5,7) → 35

const isBig = isFirstBig(5, 12);
console.log(isBig) // false
Explanation: 5 > 12 → false

Output (Example)
- add2(17,13) → 30
- multiply(5,7) → 35
- sumAll(1,2,3,4,5,6) → 21
- isFirstBig(5,12) → false
- doMath(3,4) → (3*2 + 4*2) = 14

Summary
- Arrow function → ES6 এ আসা নতুন syntax।
- Function declaration → পুরনো style।
- Function expression → variable এ function assign।
- Arrow function → ছোট, clean, modern।
- Multiple parameters → সহজে handle করা যায়।
- Multi-line arrow function → {} এবং return ব্যবহার করতে হয়।
- Module 1-4 → Arrow function এর ব্যবহার, multiple parameters, function body।

Module 1-5: More Arrow Functions and Arrow Functions (=>)

Cue (কী-ওয়ার্ড / প্রশ্ন)
- Arrow function কীভাবে normal function থেকে আলাদা?
- Arrow function লেখার ৩টি সাধারণ উপায় কী?
- Single parameter arrow function কিভাবে লেখা হয়?
- Anonymous arrow function কোথায় ব্যবহার হয়?

Notes (বিস্তারিত তথ্য)

1. Arrow Function vs Normal Function
- Normal function → function keyword দিয়ে লেখা হয়।
- Arrow function → ES6 এ আসা নতুন syntax, ছোট এবং clean।
- Arrow function এ "this" আলাদা behave করে (advanced topic)।
- Arrow function সাধারণত ছোট কাজের জন্য ব্যবহার হয়।

2. Arrow Function লেখার ৩টি উপায়
- Multiple parameters → const add = (p, q) => p + q;
- Single parameter → const getHalf = num => num / 2;
- No parameter → const logIt = () => console.log(78);

Code Example + Explanation

// Example: Arrow function with multiple parameters
const add = (p, q) => p + q;
Explanation: দুইটা parameter নিয়ে যোগ করছে। Output হবে p+q।

// Example: Arrow function with single parameter
const getSquare = (x) => x * x;
Explanation: একটাই parameter x নিয়ে তার square করছে।

const getHalf = num => num / 2;
Explanation: একটাই parameter num নিয়ে তার half করছে। এখানে () বাদ দেওয়া হয়েছে কারণ single parameter।

const firstElement = nums => nums[0];
Explanation: array এর প্রথম element return করছে।

const logIt = () => console.log(78);
Explanation: কোনো parameter নেই। শুধু console এ 78 print করছে।

---

// Function calls
console.log(getSquare(8)) // 64
Explanation: 8*8 = 64

console.log(getHalf(18)) // 9
Explanation: 18/2 = 9

console.log(firstElement([29, 65, 47])) // 29
Explanation: array এর প্রথম element 29 return করছে।

console.log(logIt()) // 78
Explanation: logIt function শুধু console এ 78 print করছে।

---

// Anonymous arrow function (event listener এ ব্যবহার)
document.getElementById('btn-click').addEventListener('click', event => {
   // এখানে event handle করা হবে
});
Explanation: এখানে কোনো নাম ছাড়া arrow function ব্যবহার হয়েছে।
এটা anonymous function, সাধারণত event handler বা callback এ ব্যবহার হয়।

Output (Example)
- add(5,7) → 12
- getSquare(8) → 64
- getHalf(18) → 9
- firstElement([29,65,47]) → 29
- logIt() → 78 (console এ print হবে)

Summary
- Arrow function → ES6 এ আসা নতুন syntax।
- Normal function → function keyword দিয়ে লেখা হয়।
- Arrow function লেখার ৩টি উপায়:
  → Multiple parameters: (p,q) => p+q
  → Single parameter: num => num/2
  → No parameter: () => console.log(78)
- Anonymous arrow function → event listener বা callback এ ব্যবহার হয়।
- Module 1-5 → Arrow function এর আরও উদাহরণ এবং ব্যবহার।


Module 1-6: Spread Operator, Array Max, Copy Arrays

Cue (কী-ওয়ার্ড / প্রশ্ন)
- Spread operator কী?
- Math.max কিভাবে array থেকে সর্বোচ্চ মান বের করে?
- কিভাবে array copy করা যায়?
- Spread operator দিয়ে object merge কিভাবে হয়?

Notes (বিস্তারিত তথ্য)

1. Spread Operator Concept
- Spread operator হলো তিনটা dot (...)  
- এর কাজ → array বা object কে আলাদা আলাদা element বা property তে ভেঙে ফেলা।
- ব্যবহার:
  → array এর element আলাদা করা।
  → array copy করা।
  → array merge করা।
  → object merge করা।
  → function এ argument পাঠানো।

2. Math.max with Spread
- Math.max() → একাধিক সংখ্যা থেকে সর্বোচ্চ মান বের করে।
- Array দিলে সরাসরি কাজ করে না।
- Spread operator দিয়ে array কে আলাদা করে পাঠালে কাজ করে।

3. Copy Arrays
- Direct assign করলে reference copy হয় (একই array modify হয়)।
- Spread operator দিয়ে copy করলে নতুন array তৈরি হয়।

Code Example + Explanation

const numbers = [87, 118, 5, 91]
console.log(numbers); // [87, 118, 5, 91]
console.log(...numbers); // 87 118 5 91
Explanation: ...numbers → array কে আলাদা আলাদা মানে ভেঙে দিয়েছে।

const max = Math.max(...numbers)
console.log(max) // 118
Explanation: Math.max(...numbers) → array এর সর্বোচ্চ মান বের করেছে।

---

// Wrong way (reference copy)
const first = [1, 2, 3, 4, 5];
const second = first;
second.push(6);
console.log(first) // [1, 2, 3, 4, 5, 6]
Explanation: second আসলে first এর reference। তাই first ও পরিবর্তিত হয়েছে।

---

// Correct way (spread copy)
const first = [1, 2, 3, 4, 5];
const second = [...first];
const third = [0, ...first, 97, 45, 12];
second.push(6);
console.log(first) // [1, 2, 3, 4, 5]
console.log(third) // [0,1,2,3,4,5,97,45,12]
Explanation: 
- second = [...first] → নতুন array তৈরি হয়েছে।
- third = [0, ...first, 97, 45, 12] → নতুন element যোগ করে merge করা হয়েছে।

---

const ages = [45, 45, 1]
const prices = [98, 65, 45]
const all = [...ages, 5555, ...prices]
console.log(all)
// [45, 45, 1, 5555, 98, 65, 45]
Explanation: দুইটা array merge হয়েছে এবং মাঝে 5555 যোগ হয়েছে।

---

const person = { name: 'Parmisan', age: 25 }
const employee = { designation: 'dev', ...person }
console.log(employee) 
// { designation: 'dev', name: 'Parmisan', age: 25 }
Explanation: object merge হয়েছে। designation property এর সাথে person এর property যোগ হয়েছে।

---

const total = (a, b, c) => a + b + c;
const result = total(45, 65, 21)
const digits = [78, 55, 66]
console.log(result); // 131
console.log(digits); // [78, 55, 66]
console.log(total(...digits)) // 199
Explanation:
- total(45,65,21) → 131
- total(...digits) → 78+55+66 = 199
- এখানে spread operator দিয়ে array কে আলাদা করে function এ পাঠানো হয়েছে।

Output (Example)
- ...numbers → 87 118 5 91
- Math.max(...numbers) → 118
- Copy array → [...first]
- Merge array → [...ages, 5555, ...prices]
- Merge object → {...person, designation:'dev'}
- Function call → total(...digits) → 199

Summary
- Spread operator (...) → array/object কে ভেঙে আলাদা করে।
- Math.max(...array) → array থেকে সর্বোচ্চ মান বের করে।
- Array copy → [...array] দিয়ে নতুন copy তৈরি হয়।
- Array merge → একাধিক array কে একসাথে যোগ করা যায়।
- Object merge → একাধিক object কে একসাথে যোগ করা যায়।
- Function call → array কে spread করে argument হিসেবে পাঠানো যায়।
- Module 1-6 → Spread operator এর ব্যবহার, array max, copy arrays।


Module 1-7 (Advanced): Object and Array Destructuring

Cue (কী-ওয়ার্ড / প্রশ্ন)
- Destructuring কী?
- Object destructuring কিভাবে কাজ করে?
- Array destructuring কিভাবে কাজ করে?
- Default value, alias, deep property, dynamic property name কীভাবে ব্যবহার হয়?

Notes (বিস্তারিত তথ্য)

1. What is Destructuring?
- Destructuring হলো ES6 এর একটি feature।
- এর মাধ্যমে object বা array থেকে value দ্রুত বের করা যায়।
- কোড ছোট এবং clean হয়।
- Function parameter এ সরাসরি value ব্যবহার করা যায়।
- Default value set করা যায়।

Object Destructuring Examples

const user = {
  name: "Hidayt",
  city: "Delhi",
  type: "Admin"
}

const { name, city, type } = user;
console.log(name, city, type) // Hidayt Delhi Admin
Explanation: object user থেকে name, city, type property আলাদা করে variable এ রাখা হয়েছে।

---

const user = {
  id: 42,
  user_name: 'dangtrunganh',
  year: 1985,
  is_verified: true
};

const { user_name: u, is_verified: iv, salary: s } = user;
console.log(u, iv, s) // dangtrunganh true undefined
Explanation:
- user_name → u নামে alias হয়েছে।
- is_verified → iv নামে alias হয়েছে।
- salary property নেই, তাই s = undefined।

---

const myObject = { student: 'Mike', teacher: 'Susan' };

const { student: pupil, teacher: prof } = myObject;
console.log(pupil, prof) // Mike Susan
Explanation: student → pupil নামে alias হয়েছে, teacher → prof নামে alias হয়েছে।

const { student, teacher } = myObject;
console.log(student, teacher) // Mike Susan
Explanation: সরাসরি property নাম ব্যবহার করা হয়েছে।

---

const employee = {
  id: 007,
  name: 'James',
  dept: 'Spy'
}

const { name, age = 25 } = employee;
console.log(name, age) // James 25
Explanation: employee এ age নেই, তাই default value 25 ব্যবহার হয়েছে।

---

const { price, quantity, tax = 7 } = { name: 'shirt', price: 500, quantity: 7 }
console.log(price, quantity, tax) // 500 7 7
Explanation: tax property না থাকলে default 7 ব্যবহার হয়েছে।

---

const device = { name: 'phone', brand: 'samsung', price: 17000 }
const { brand } = device;
console.log(brand) // samsung
Explanation: শুধু brand property destructure করা হয়েছে।

Array Destructuring Examples

const numbers = [25, 88, 89, 101]
const [first, second] = numbers;
console.log(first, second) // 25 88
Explanation: array এর প্রথম দুইটা মান আলাদা করা হয়েছে।

const [math, physics] = [90, 99];
console.log(physics) // 99
Explanation: array থেকে সরাসরি মান destructure করা হয়েছে।

Output (Example)
- Object destructuring → { name, city } = user
- Alias → { user_name: u } = user
- Default value → { age = 25 } = employee
- Array destructuring → [first, second] = numbers
- Dynamic property → { [propName]: myProp } = object (advanced)

Summary
- Destructuring → object/array থেকে value দ্রুত বের করার উপায়।
- Object destructuring → property কে variable এ রাখা যায়।
- Array destructuring → index অনুযায়ী মান বের করা যায়।
- Default value → property না থাকলে fallback মান ব্যবহার হয়।
- Alias → property কে অন্য নামে ব্যবহার করা যায়।
- Module 1-7 → Advanced destructuring এর ব্যবহার object এবং array এ।


Module 1-8: Keys, Values, Entries, Delete, Seal, Freeze

Cue (কী-ওয়ার্ড / প্রশ্ন)
- Object.keys(), Object.values(), Object.entries() কী করে?
- delete operator কীভাবে কাজ করে?
- Object.seal() আর Object.freeze() এর মধ্যে পার্থক্য কী?
- কোনটা কখন ব্যবহার করা উচিত?

Notes (বিস্তারিত তথ্য)

1. Object.keys()
- Object এর property নামগুলো (keys) কে array আকারে return করে।

2. Object.values()
- Object এর property এর মানগুলো (values) কে array আকারে return করে।

3. Object.entries()
- Object এর property নাম এবং মানগুলোকে [key, value] pair আকারে array return করে।

4. delete operator
- Object থেকে কোনো property remove করতে delete ব্যবহার করা হয়।

5. Object.seal()
- নতুন property add করা যায় না।
- Property delete করা যায় না।
- কিন্তু existing property এর value change করা যায়।

6. Object.freeze()
- Object পুরোপুরি freeze হয়ে যায়।
- নতুন property add করা যায় না।
- Property delete করা যায় না।
- Property এর value change করা যায় না।

Code Example + Explanation

const employee = {
    name: 'Raja Rani',
    designation: 'QA',
    salary: '20000',
    experience: 1
}

const keys = Object.keys(employee)
const values = Object.values(employee)
const entries = Object.entries(employee)

console.log(keys) 
// [ 'name', 'designation', 'salary', 'experience' ]
Explanation: Object.keys() → সব property নাম array আকারে দিয়েছে।

console.log(values) 
// [ 'Raja Rani', 'QA', '20000', 1 ]
Explanation: Object.values() → সব property এর মান array আকারে দিয়েছে।

console.log(entries)
/*
[
  [ 'name', 'Raja Rani' ],
  [ 'designation', 'QA' ],
  [ 'salary', '20000' ],
  [ 'experience', 1 ]
]
*/
Explanation: Object.entries() → প্রতিটি property কে [key, value] pair আকারে দিয়েছে।

---

const employee = {
    name: 'Raja Rani',
    designation: 'QA',
    salary: 20000,
    experience: 1,
    age: 22
}

Object.freeze(employee)
delete employee.experience
employee.salary = employee.salary + 5000;
employee.location = 'Dhaka'
console.log(employee)

Explanation:
- Object.freeze(employee) → object freeze হয়েছে।
- delete employee.experience → কাজ করবে না।
- employee.salary = employee.salary + 5000 → কাজ করবে না।
- employee.location = 'Dhaka' → নতুন property add হবে না।
Result: object অপরিবর্তিত থাকবে।

---

const employee = {
    name: 'Raja Rani',
    designation: 'QA',
    salary: 20000,
    experience: 1,
    age: 22
}

Object.seal(employee)
delete employee.experience
employee.salary = employee.salary + 5000;
employee.location = 'Dhaka'
console.log(employee)

Explanation:
- Object.seal(employee) → object seal হয়েছে।
- delete employee.experience → কাজ করবে না।
- employee.salary = employee.salary + 5000 → value change হবে।
- employee.location = 'Dhaka' → নতুন property add হবে না।
Result: শুধু existing property এর value পরিবর্তন করা যাবে।

Output (Example)
- Object.keys(employee) → [ 'name', 'designation', 'salary', 'experience' ]
- Object.values(employee) → [ 'Raja Rani', 'QA', '20000', 1 ]
- Object.entries(employee) → [ ['name','Raja Rani'], ['designation','QA'], ... ]
- Object.freeze(employee) → add/delete/change কিছুই করা যাবে না।
- Object.seal(employee) → add/delete করা যাবে না, কিন্তু value change করা যাবে।

Summary
- Object.keys() → property নাম array আকারে।
- Object.values() → property মান array আকারে।
- Object.entries() → [key, value] pair array আকারে।
- delete → property remove করতে ব্যবহার হয়।
- Object.seal() → add/delete করা যায় না, value change করা যায়।
- Object.freeze() → add/delete/change কিছুই করা যায় না।
- Module 1-8 → Object এর keys, values, entries, delete, seal, freeze এর ব্যবহার।



Module 1-9: Loop through an Object using for...in

Cue (কী-ওয়ার্ড / প্রশ্ন)
- for...of loop কীভাবে কাজ করে?
- for...in loop কীভাবে কাজ করে?
- Object.keys() দিয়ে loop করার সুবিধা কী?
- ES6 এর সবচেয়ে common features কী কী?

Notes (বিস্তারিত তথ্য)

1. for...of Loop
- Array বা iterable এর উপর loop চালাতে ব্যবহার হয়।
- প্রতিটি element একে একে পাওয়া যায়।

2. for...in Loop
- Object এর property নাম (keys) এর উপর loop চালাতে ব্যবহার হয়।
- প্রতিটি key এর সাথে value access করা যায়।

3. Object.keys() + for...of
- Object.keys() → object এর সব property নাম array আকারে দেয়।
- তারপর for...of দিয়ে সেই array এর উপর loop চালানো যায়।
- এর মাধ্যমে object এর value পাওয়া যায়।

Code Example + Explanation

const numbers = [1, 2, 54, 1, 4, 9];
for(const num of numbers){
    console.log(num);
}
Explanation: for...of loop array এর প্রতিটি মান একে একে print করছে।

---

const employee = {
    name: 'Raja Rani',
    designation: 'QA',
    salary: 20000,
    experience: 1,
    age: 22
}

for(const key in employee){
    const value = employee[key];
    console.log(key, value);
}
Explanation:
- for...in loop object এর প্রতিটি property নাম (key) বের করছে।
- employee[key] দিয়ে সেই key এর value পাওয়া যাচ্ছে।
- Output: name Raja Rani, designation QA, salary 20000, experience 1, age 22

---

const keys = Object.keys(employee);
console.log(keys)
// [ 'name', 'designation', 'salary', 'experience', 'age' ]

for(const key of keys){
    const value = employee[key];
    console.log(key, value);
}
Explanation:
- Object.keys(employee) → সব property নাম array আকারে দিয়েছে।
- for...of loop দিয়ে সেই array এর উপর loop চালানো হয়েছে।
- প্রতিটি key এর value পাওয়া গেছে।

Output (Example)
- for...of → array এর প্রতিটি মান পাওয়া যায়।
- for...in → object এর প্রতিটি property নাম পাওয়া যায়।
- Object.keys() + for...of → object এর property নাম array আকারে নিয়ে loop চালানো যায়।

Module Summary: Common ES6 Features
- let এবং const → নতুন variable declare করার জন্য।
- Arrow function (=>) → ছোট এবং clean function syntax।
- Template literals (` `) → multiple line string এবং dynamic string।
- Default parameters → function এ argument না দিলে fallback মান।
- Spread operator (...) → array/object copy, merge, এবং function argument।
- Destructuring → object/array থেকে value দ্রুত বের করা।
- for...of এবং for...in → array এবং object এর উপর loop চালানো।
- Object methods → Object.keys(), Object.values(), Object.entries(), seal(), freeze()।

Summary
- for...of → array এর উপর loop।
- for...in → object এর উপর loop।
- Object.keys() → object এর property নাম array আকারে।
- ES6 → modern JavaScript এর জন্য অনেক নতুন feature দিয়েছে।
- Module 1-9 → object loop করার উপায় এবং ES6 এর common features।
